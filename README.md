## Description  
This project models insurance risk and pricing using a structured dataset of customers, policies, and claims. It estimates expected losses by analysing claim frequency and severity, and evaluates portfolio performance under different scenarios.

---

## Features  
- Relational database design (customers, policies, claims)  
- Data validation and integrity checks  
- Claim frequency and severity analysis  
- Loss ratio calculations (claims / premium)  
- Feature engineering (frequency, severity, exposure)  
- Statistical pricing model (frequency × severity)  
- Risk segmentation by customer attributes  
- Advanced SQL analytics (window functions, ranking)  
- Scenario simulation (frequency shocks, large losses)  
- Visualisation of claims and risk metrics  

---

## Pricing Model  
- Expected Loss = Frequency × Severity  
- Poisson model for claim frequency  
- Gamma / Lognormal model for claim severity  
- Segment-based risk pricing
